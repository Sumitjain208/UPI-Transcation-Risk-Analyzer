# Smart UPI Transaction Risk Analyzer

A Machine Learning based web application that analyzes UPI transactions and predicts whether a transaction is risky or safe using Random Forest Classifier and Streamlit.

## Features

- UPI transaction risk prediction
- Risk score generation
- Interactive Streamlit dashboard
- Real-time transaction analysis
- Machine learning model integration


## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib
- Seaborn

## Machine Learning Algorithm

- Random Forest Classifier

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Data Preprocessing
5. Model Training
6. Model Evaluation
7. Streamlit App Development
8. Deployment

## Dataset Features

- Transaction Amount
- Transaction Status
- Transaction Hour
- Day
- Month
- Night Transaction
- Sender Bank
- Receiver Bank
